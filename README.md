# Hogwarts Project

## Part 1 – Student overview

This part of the project displays Hogwarts students using data from the HP API.

Students are shown as cards with:
- image (default image if missing)
- name
- alternative names
- age
- wand information
- house

The user can:
- search students by name or house
- filter students by Hogwarts house
- sort students by age
- add a new student using a form
- edit student name, house and year of birth
- delete students from the list

There is also a save feature:
- user can save up to 3 favorite students
- saved students are stored in localStorage
- saved students are shown in a separate list

Custom students are also saved in localStorage.
